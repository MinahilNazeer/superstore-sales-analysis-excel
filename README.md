# Superstore Sales Analysis (Excel)

## Project Overview

This project analyzes the Superstore dataset using Microsoft Excel to answer key business questions regarding sales performance, profitability, customer behavior, regional performance, and the impact of discounts. The project demonstrates the use of Excel for business intelligence through data cleaning, Pivot Tables, Pivot Charts, KPI calculations, dashboard creation, and executive reporting.

## Data Source

- **Dataset:** Sample - Superstore
- **Source:** [Tableau Sample Data](https://www.kaggle.com/datasets/vivek468/superstore-dataset-final)
- The dataset contains customer orders, product details, sales, profits, discounts, shipping information, and regional data for a fictional retail company.

## Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Excel Formulas
- Dashboard Design

## Business Questions

- Which category generates the highest sales?
- Which products are unprofitable?
- Which regions underperform?
- Do discounts reduce profit?
- Who are the most valuable customers?

## Key Findings

- Technology generated the highest sales and profit.
- Bookcases, Tables, and Supplies generated losses.
- West region produced the highest profit.
- Central region underperformed.
- Higher discounts generally reduced profitability.
- Consumer was the highest revenue-generating customer segment.

## Recommendations

- Reduce excessive discounts on low-profit products.
- Review pricing strategies for loss-making categories.
- Investigate operational issues in underperforming regions.
- Focus on retaining high-value customers.
- Continue investing in Technology products.
